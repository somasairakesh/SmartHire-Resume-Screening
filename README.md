# SmartHire - Resume Screening System

## 📌 Overview
SmartHire is a simple machine learning project that ranks resumes based on their similarity to a job description.

---

## 🚀 Task 1 Completed
- Converted text into numerical vectors using TF-IDF
- Compared resumes using cosine similarity
- Generated similarity scores

---

## 🛠️ Tech Stack
- Python
- scikit-learn

---

## 📊 Sample Output

Resume 2 Score: 0.22
